### Added

## [0.3.0] - 2015-11-20 (Pulled)
### Added
- stub for metrics-couchbase.py

## [0.2.0] - 2015-11-20

- add couchbase cluster check
- add couchbase bucket replica check
- metrics-couchbase.py: add ep_cache_miss_rate metric
